Oefen Frans in Parijs

Deze app is nu een Python Streamlit applicatie!

🚀 Aan de slag

1
Installeer Streamlit
pip install streamlit
2
Start de app
streamlit run src/webapp.py
3
Leer Frans!
Klik op een locatie op de kaart en beantwoord de vragen.
💡 Tip: De app opent automatisch in je browser op localhost:8501
